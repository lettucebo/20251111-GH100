# GH-100 課程與範例 DEMO

## 📚 課程簡介

這是 **GH-100** 課程的範例專案與示範程式碼倉庫。本專案旨在提供學習者一個完整的實作環境，透過實際的範例來深入理解 GitHub 的核心功能與最佳實踐。

## 🎯 課程目標

本課程與範例 DEMO 將幫助您：

- **掌握 GitHub 基礎操作**：學習如何建立、管理和維護 GitHub 倉庫
- **版本控制實務**：深入了解 Git 版本控制系統的使用方法
- **協作開發流程**：掌握 Pull Request、Issue、Code Review 等協作工具
- **自動化工作流程**：學習使用 GitHub Actions 建立 CI/CD 流程
- **專案管理技巧**：運用 GitHub Projects 進行專案規劃與追蹤

## 📋 專案內容

本範例 DEMO 包含以下內容：

### 基礎操作範例
- Repository 建立與設定
- Branch 管理與合併策略
- Commit 訊息規範

### 協作開發範例
- Fork 與 Pull Request 工作流程
- Issue 管理與追蹤
- Code Review 最佳實踐

### 進階功能範例
- GitHub Actions 自動化範例
- GitHub Pages 部署
- 專案文件撰寫

## 🚀 快速開始

### 前置需求

- Git 已安裝在您的電腦上
- 擁有 GitHub 帳號
- 基本的命令列操作知識

### 開始使用

1. **複製此倉庫**
   ```bash
   git clone https://github.com/lettucebo/20251111-GH100.git
   cd 20251111-GH100
   ```

2. **瀏覽範例內容**
   
   探索專案中的各個範例檔案與說明文件

3. **實際操作練習**
   
   跟隨課程指引，實作各項 GitHub 功能

歡迎提出問題、建議或貢獻程式碼！請遵循以下步驟：

1. Fork 此專案
2. 建立您的功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的變更 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 開啟一個 Pull Request
